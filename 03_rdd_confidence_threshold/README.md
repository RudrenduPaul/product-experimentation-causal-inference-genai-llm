# 03 — Regression Discontinuity Design for LLM Confidence-Threshold Routing

Companion code for FCC Article 3: *"Product Experimentation with Regression Discontinuity: Causal Inference for LLM Confidence-Threshold Routing in Python"*.

## What this teaches

Measuring the causal effect of routing low-confidence LLM queries to a premium model when the routing rule creates a sharp threshold experiment. Covers:

- Why the routing threshold at confidence = 0.85 functions as a natural experiment
- Sharp RDD with local linear regression and HC3 heteroskedasticity-robust standard errors
- Bandwidth sensitivity sweep across four window sizes (0.05, 0.10, 0.15, 0.20)
- McCrary-style density check for manipulation near the threshold
- Quadratic robustness specification to test whether the linear fit is adequate
- Bootstrap 95% confidence intervals for every point estimate

## Run

```bash
# From the repo root
python data/generate_data.py --seed 42 --n-users 50000 --out data/synthetic_llm_logs.csv
```
